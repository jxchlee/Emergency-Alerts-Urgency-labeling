# 한국 재난 문자 데이터셋 (Korean Emergency Alerts Dataset)

이 데이터는 [재난 안전 데이터 공유 플랫폼](https://www.safetydata.go.kr/disaster-data/disasterNotification) 을 크롤링하여 응급도를 라벨링한 데이터입니다.

2023/09/01 ~ 2025/04/18 까지의 데이터 중 랜덤한 데이터 1000건을 골라 라벨링을 진행하였습니다. 재난 문자의 내용을 분석하거나, 긴급한 재난 상황과 단순 알림(실종 등)을 구분하는 모델을 학습하는 데 활용할 수 있습니다.

## 데이터셋 개요

- **파일 이름**: `Emergency_Alerts.csv`
- **인코딩**: UTF-8-SIG
- **데이터 크기**: 약 190KB

### 컬럼 설명 (Columns)

데이터셋은 총 8개의 컬럼으로 구성되어 있습니다.

| 컬럼명 (Column) | 설명 (Description) | 예시 |
|-----------------|--------------------|------|
| `create_date` | 문자 수신 날짜 (YYYY-MM-DD) | `2024-08-06` |
| `create_time` | 문자 수신 시간 (HH:MM:SS) | `14:02:35` |
| `location_name` | 광역/지역 명칭 | `경기도`, `부천시` |
| `sender` | 발송 기관 (지자체, 경찰청 등) | `부천시청`, `충남경찰청` |
| `msg_type` | 재난 종류 (추정) | `폭염`, `실종` |
| `md101_sn` | 일련번호 | `7`, `13` |
| `msg` | 재난 문자 본문 내용 | `[부천시청] 오늘 14:00 폭염경보 발효...` |
| `urgency_label` | **긴급도 라벨**  | `1`, `2`, `3`, `4` |

### 라벨 설명 (`urgency_label`)

- 1: 실제로 일어나지 않은 단순한 안내문자, 예방. 실종, 산불 예방
    - 당진시 주민인 김재원씨(남,25세)를 찾습니다-165cm,78kg,회색반팔티(아하방수택작업복),검정체육복바지 vo.la/DGHFv / ☎182 [충남경찰청]
    - 물놀이 안전사고 예방을 위해 안전수칙 준수바랍니다. ▲물놀이위험구역 접근금지▲충분한 준비운동▲구명조끼 착용▲음주수영·야간입수 금지▲어린이는 보호자와 동행 [서구]

- 2: 주의가 필요하거나 2회차 이상의 반복되는 경보/주의보 문자
    - 강우, 강설 및 기온 강하로 인해 도로 결빙이 우려되니 출근길 차량 운전 시 서행 및 거리 유지, 보행자 낙상사고 주의 등 안전에 유의 바랍니다.[창녕군]
    - 폭염이 연일 지속되어 온열질환 발생 위험이 높으니 야외활동 및 야외작업(논,밭,건설현장 등)을 자제하시고 그늘,휴식,수분섭취 등 건강관리에 유의바랍니다.[여주시]

- 3: 직접적인 피해가 예상되는 문자. 기상 특보 등
    - 4.12. 18시 부산지역에 강풍주의보가 발효되었습니다. ▲낙하 위험물 고정▲외출 자제▲해안가 접근 자제 등 안전사고에 유의바랍니다.[부산시]
    - [부산진구청] 금일 00시 50분경 범천동 1197-3 인근에 멧돼지 1마리가 출몰했다는 신고가 접수되었으니 인근 주민께서는 안전에 유의하시기 바랍니다.

- 4: 즉시 대피해야하는 문자
    - 영암군 학산면 신안마을 침수 진행중▲신안마을 주민분들께서는 인근 마을회관으로 신속히 대피하여 주시기 바랍니다. [영암군]
    - (대피명령 발령) 산불 확산으로 단촌면 장림리 주민과 등산객은 단촌초등학교로 대피하시기바랍니다.  [의성군]
